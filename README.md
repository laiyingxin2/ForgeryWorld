## 📚 Three Routes

### Route 1: Internal-article port (Baseline #1)

```bash
python orchestrator.py --mode v1 --briefs 25 --rollouts 4 \
    --api-only-ops --out outputs_m1 --src-pool data/real_faces/*.png
```

Composition: SimpleFamilySelector (random) + SimpleSkillBook (single markdown) + SimpleMCTSPlanner (UCB1 4-step) + 6 mutation operators + multi_agent_gen (6 LLM roles). **No self-evolution learning**.

### Route 2: Paper-based + my improvements (Baseline #2)

```bash
python orchestrator.py --mode v2 --briefs 15 --rollouts 2 \
    --api-only-ops --src-pool data/real_faces/*.png
```

10-layer architecture:

| Layer | Module | Mechanism |
|---|---|---|
| L0 env | `operators/api_image.py` | 15 attack ops (10 local + 5 API) |
| L1 family | `markov_family.py` | MAJIC Markov transition (paper 2508.13048) |
| L2 multi-agent | `multi_agent_gen.py` | 6-LLM 出题/质检/主管 |
| L3 skill | `ace_skill_lib.py` | Ace-Skill Eq.4/7/8 (paper 2605.08887) |
| L4 lookahead | `pipeline_planner.py` | WebEvolver k=3 d=2 (2504.21024) |
| L5 sandbox | `sandbox.py` | 3-tier verifier (Tier-1 num + Tier-2 LLM + Tier-3 cross-check) |
| L6 attribution | `self_attributor.py` | AgentEvolver SA Eq.20 (2511.10395) |
| L7 data flow | `data_flow.py` | UI-TARS-2 V=1→SFT/V=0→CT (2509.02544) |
| L8 memory | (L1-L5 hierarchy) | Designed but not implemented |
| L9 diagnose | `diagnosis_and_export.py` | Agent-World diagnose (2604.18292) |
| L10 defender | `diagnosis_and_export.py` | Qwen2.5-VL SFT export |

**Plus my improvements** (none of 6 cloned repos had these):
- `reflexion.py` — intra-rollout verbal reflection (Shinn NeurIPS 2023)
- `reasoning_bank.py` — parallel strategy-rule stream (Xu Google 2509.25140)
- `novelty.py` — anti mode-collapse novelty term

### Route 3: Lv5 — connect FakeVLM for model self-evolution

```bash
# A. Export attacker + defender SFT pools from accumulated trajectories
python launch_lv5.py --phase prep

# B. Deploy FakeVLM as Tier-2 detector via vLLM (GPU required, 18GB VRAM)
bash outputs/lv5_deploy_vllm.sh
curl http://localhost:8000/health  # check up

# C. Train FakeVLM on new attack samples (LoRA SFT, 1-6h on H100)
bash outputs/lv5/attacker_lv5/lv5_attacker_train.sh

# D. Eval: bypass rate vs gemini Tier-2 (pre) vs trained FakeVLM Tier-2 (post)
python orchestrator.py --mode v2 --tier2-backend viviai \
    --briefs 15 --rollouts 2 --out outputs_pre_lv5
python orchestrator.py --mode v2 --tier2-backend fakevlm_local \
    --briefs 15 --rollouts 2 --out outputs_post_lv5
python analyze_runs.py --m1 outputs_pre_lv5 --m2 outputs_post_lv5
```

Lv5 uses **existing FakeVLM weights** (13.4 GB safetensors at
`/cpfs01/bob_workspace/students/lyx/ICML/FakeVLM/Origin/FakeVLM/checkpoints/multi_20260329_132526_llava-1.5-7b`),
**not** from-scratch Qwen2.5-VL. Recipe = SEAL (MIT 2506.10943) + STaSC small-data (2503.08681) + RISE per-step credit (2407.18219).

## 🐛 Bug Fix Log (15 silent failures found)

### Round 1 (6 bugs)
1. **Skill not accumulating** — `random.random() < 0.3` gate too tight, almost never fired → `supervisor_every` explicit config
2. **trajectory_id collision across runs** — `INSERT OR REPLACE` silent overwrite → run_id prefix
3. **Markov matrix reset every init** → save/load JSON
4. **defender_sft.jsonl "w" mode** → "a" mode (append cross-run)
5. **bypass_stats / markov_snapshots PK collision** → PK + run_id
6. **recent_briefs/outcomes + total_cost reset** → meta state persist

### Round 2 (4 bugs)
7. **Q1: regex `[01]?\.\d+`** fails on `"1"` / `"85%"` → broader regex + % support
8. **Q4: 3 new modules imported but never called in orchestrator** → integrate at init + run_one_rollout
9. **Q9: lv5_connector uses src_face_path (raw input)** for SFT image → use exec_steps[-1].output_path (attack product)
10. **Q12: meta save only at round end** → atomic_write helper

### Round 3 (5 bugs)
11. **Q14: Reflexion not given available_ops** → 100% suggested invented ops → inject ops list
12. **Q17: tier2 call fails → success=False, default is_fake=False, conf=0.5 → sandbox_pass=True false positive** → require `success=True`
13. **Q18: parse_json_robust written but unused** → integrate in multi_agent / self_attributor / reasoning_bank / reflexion / viviai_client (5 places)
14. **V1A: method 1 SimpleSkillBook also doesn't load** → fix v1 init load_all
15. **viviai 503 / nano_banana 503 no retry / fallback** → 5× exp-backoff + pro→2→1 fallback chain

### General guards added (from search agent)
16. **`$47K stall loop`** → CostBudget + DuplicateCallDetector
17. **`SQLite WAL`** → `enable_wal_mode()` helper
18. **`Atomic write`** → tmp+os.replace+fsync
